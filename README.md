# Civilization Cognitive Ecosystem

Civilization Cognitive Ecosystem（CCE）は、文明における知を情報量ではなく構造として扱い、  
その文明内での現れ、接続、再配置、泡沫化、再発見を、生態系的かつ動的な構造場として記述する理論である。

CCEにおいて、構造そのものは世界に元から存在する。  
CCEが扱うのは、その構造が文明内でどのように観測され、接続可能化され、再配置され、定着し、あるいは泡沫化するかという過程である。

---

## 要旨

文明を「知識の集積」として捉えるのではなく、  
「知という構造が、媒体・個人・集団・時間をまたいで保存・変形・再接続され続ける生態系」として捉える。

CCEの中核的な再定義は、**知 = 構造** である。

これにより、文明における中心問題は次のように移る。

- 「何が知られているか」ではなく、「何が構造として残るか」
- 「何が別媒体へ写し替えられるか」
- 「何が別領域へ移植されたときに壊れ、何が残るか」

CCEの本体は星図（構造の分布）ではなく、**気候**（何が見え、何が接続され、どこで渦が発生しやすいかを変化させる文明流体システム）である。

---

## 立場

CCEは次の立場を取る。

1. 構造は世界に元から存在する。CCEが扱うのは構造の創出ではなく、文明内での現れと変形である
2. 知を情報量ではなく、関係の形として扱う
3. 文明を国家や時代の単位に閉じず、都市、儀礼、技術、GitHub、対話、音楽、師弟関係といった多様な場を含む構造場として扱う
4. 主体を内部要素として含まない。人間やAIは、圧、作用、観測装置、越境エージェントとして外部的に表現される

---

## 非目的

CCEは、以下を直接の目的としない。

- 知識量の測定や比較
- 特定の文明の優劣評価
- AIの知性の定義
- 教育理論や学習理論の提供
- 百科事典的な構造の網羅

---

## 中核概念

### 構造

複数の要素が一定の関係を持つことで現れる、反復可能な関係のまとまり。  
CCEでは知を内容としてではなく、関係の形として扱う。

### 観測

潜在していた構造が文明内部で認識され、接続可能な構造として現れること。  
観測は単なる認識ではなく、接続可能化を含む出来事である。

### 重力

構造そのものに宿る汎用性。  
異なる文脈やドメインでも形を保ち、越境後も壊れにくい度合い。  
重力の高い構造は古典化し、基盤となる。

### 気候

何が見え、何が接続され、どこで渦が発生しやすいかを変化させる文明流体システム。  
CCEの本体は星図ではなく気候である。

### 越境

構造がドメインやメタドメインを跨いで配置されること。  
越境後の定着には文脈補完要求が立ち上がる。

### 渦

構造の再配置と接続変化が局所的に集中する現象。  
渦は構造を無から生み出すものではなく、既に存在していた構造の顕在化、再配置、接続変化の局所集中を担う。

### 潜在相

星になりうる構造の状態。  
領域ではなく状態であり、真暗黒領域にも、星団縁辺にも、既知領域内部の空白部にも現れうる。

## 世界構造とCCEの位置

CCEは、より大きな層構造の上位に置かれる。

```
RL∞（世界法則）
↓
数学（地殻）
↓
自然科学（観測準備層）
↓
言語（土壌）
↓
CCE（文明生態系）
```

---

## 関連理論

CCEは、以下の理論と隣接するが、同一ではない。

- **逆説的AGI** — CCEの下位層であるRL∞を共有する。逆説的AGIは対話における分岐と摩擦の配置を扱い、CCEは文明スケールでの構造の分布と変形を扱う
- **主体影理論** — SSTの第3論文（文化／環境／開放系）とCCEの気候・越境は射程が重なりうる。ただしSSTは主体知覚を扱い、CCEは構造の動態を扱う
- **Relative Persona Model** — RPMの背景記憶の変質（Knowledge Background → Lived Background）は、CCEにおける構造の観測・定着と構造的に対応しうる
